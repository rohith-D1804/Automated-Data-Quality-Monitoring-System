# Automated Data Quality Monitoring System

## Problem Statement

Organizations often receive datasets containing:
- Missing values
- Duplicate records
- Outliers
- Schema inconsistencies

Manual validation is time-consuming.

This project automates the process of validating datasets and generating quality reports.

---

## MVP Features

1. CSV Upload
2. Missing Value Detection
3. Duplicate Detection
4. Outlier Detection
5. Schema Validation
6. Quality Score Calculation
7. Dashboard
8. Scan History
